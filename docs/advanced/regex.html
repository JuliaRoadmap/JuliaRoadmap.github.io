    正则表达式

    泛使用

    regex

    julia> r=r"a|b"
    r    "a|b"

    julia> findall(r,"abc")
    2-element Vector{UnitRange{Int64}}:
         1:1
         2:2

    该链接中的标志/模式修正符对应第二个参数

    julia> r=Regex("a|b","i")
    r    "a|b"i

    julia> findall(r,"AB")
    2-element Vector{UnitRange{Int64}}:
         1:1
         2:2

    运算

    julia> r"a|b"==r"b|a" # 不会区分
    false

    julia> r"a"*r"b"
    r    "(?:a)(?:b)"

    julia> r"a"*"b"
    r    "(?:a)\Qb\E"

    julia> r"a"^2
    r    "(?:a){2}"
